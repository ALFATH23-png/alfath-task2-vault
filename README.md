⭐ 8. PERT ANALYSIS (PROGRAM EVALUATION AND REVIEW TECHNIQUE)
(Distinction Level, Case Study Specific)
PERT analysis is used to estimate realistic task durations by calculating the weighted average of:
•	Optimistic time (O) — if everything goes perfectly
•	Most likely time (M) — normal expected duration
•	Pessimistic time (P) — if delays occur
The formula is:
[ \text{PERT Time} = \frac{O + (4 \times M) + P}{6} ]
This gives a more accurate prediction than guessing or using a single number.
For your project, PERT is essential because:
•	recruitment depends on availability of 18+ players
